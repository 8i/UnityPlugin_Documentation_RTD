Platform Support
============================================================

Both codecs are supported on mobile targets, however, PAT only supports in-editor preview on Linux.

+-----------+-----------+----------+
| Platform  | HVR       | PAT      |
+-----------+-----------+----------+
| Android   | Yes       | Yes      |
+-----------+-----------+----------+
| iOS       | Yes       | Yes      |      
+-----------+-----------+----------+
| Linux     | Yes       | Yes      |
+-----------+-----------+----------+
| MacOS     | Yes       | No       |
+-----------+-----------+----------+
| Windows   | Yes       | No       |
+-----------+-----------+----------+
